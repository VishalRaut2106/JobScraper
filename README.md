# Job Scraper Bot

A daily automation tool that searches for "Software Engineer" and "Internship" roles using a custom scraper (DuckDuckGo HTML) and sends alerts via Telegram and Email.

## Features
- **Zero Cost**: Uses free GitHub Actions.
- **No API Keys needed for Search**: Uses HTML scraping.
- **Notifications**: Telegram & Gmail.

## Setup
Add these Secrets to your Repo:
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
- `EMAIL_USER`
- `EMAIL_PASS`
